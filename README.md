# JNTUK Student Library

A mobile-first web application for JNTUK students to:
- Access study materials
- Calculate CGPA
- Manage academic data

## 🚀 Features
- OTP-based login (or email auth)
- Material browsing
- CGPA calculator
- Profile system

## 🛠 Tech Stack
- React
- Firebase
- Node.js (if used)
